# filterNHP

Non-human primate search filters

<!-- badges: start -->

<!-- badges: end -->

The goal of filterNHP is to is to provide users with an easy way to
obtain a comprehensive search filters to use in popular databases such
as [PubMed](https://pubmed.ncbi.nlm.nih.gov/),
[PsycInfo](http://search.ebscohost.com/Login.aspx?profile=web&defaultdb=psyh&lp=login.asp&ref=https%3A%2F%2Fwww%2Egoogle%2Ecom%2F&authtype=ip,uid)
and [Web of
Science](http://login.webofknowledge.com/error/Error?Error=IPError&PathInfo=%2F&RouterURL=http%3A%2F%2Fwww.webofknowledge.com%2F&Domain=.webofknowledge.com&Src=IP&Alias=WOK5).

## Installation

You can install the released version of filterNHP from
[CRAN](https://CRAN.R-project.org) with:

``` r
install.packages("filterNHP")
```

And the development version from [GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("avrincon/filterNHP")
```

library(filterNHP)
